COVID-19 Project Description

Covid-19 website is a source of information about Corona Virus disease. This website consists of several features that help the users to better understand this disease. They can get to know about the symptoms of the Coronavirus and some precautions that they can use to prevent this disease. They can use this information to save themselves and their love ones. Users can get information regarding the number of cases of COVID-19 around the world on this website. In brief, for each country, they can get to know about new confirmed cases, new deaths, new recovered cases, total confirmed, total deaths and total recovered cases. I have used Covid19 API to fetch the updated data and display it for the users.

Paradise Project Description

The paradise resort website act as public interface through which online users can interact with the resort and its facilities. This website includes all kinds of features where users can get to know about the activities that resort is organizing, can plan their stay at the resort, can read the blogs to better know the resort, can contact the resort for any question they want to know, can give their feedback to appreciate or request to improve the facilities of the website or resort. The users can also apply for the jobs that the resort posts on this website and if they face any difficulty in applying for a job, they can call our help centre or can ask their query by online filling the help centre form. This website also has an admin face. Admin can log in to their account and can manage the whole data of the website. Admin can add new information, update and delete the posted data of the website.

Developer and Designer

Features Worked On:

This website has many features that help the guests to book a table and select their food from the menu and can get to know about the special meal of the day.
